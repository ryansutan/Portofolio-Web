Our Profile

Nama saya Ryan Sutan Wijaya, lulusan S1 Sistem Informasi dari Universitas Trunojoyo Madura dengan IPK 3.58. Saya memiliki ketertarikan dalam pengembangan sistem informasi dan analisis data. Saya pernah magang di Dinas Kesehatan Sidoarjo dalam pengembangan aplikasi web menggunakan HTML, CSS, PHP, dan MySQL. Selain itu, saya juga memiliki pengalaman membuat visualisasi data menggunakan Power BI. Saat ini saya aktif mengikuti pelatihan dan terus mengembangkan keterampilan melalui berbagai proyek pribadi.

Website Rekomendasi Wisata

Proyek ini merupakan aplikasi web rekomendasi tempat wisata yang dibangun menggunakan PHP, CSS, dan JavaScript, serta terintegrasi dengan database MySQL. Website ini didesain dengan dua jenis pengguna, yaitu Admin dan Pengunjung.
Proyek ini tidak memberikan rekomendasi secara acak. Nantinya akan diterapkan metode SMART (Simple Multi-Attribute Rating Technique) untuk menghasilkan rekomendasi yang lebih objektif dan relevan berdasarkan kriteria tertentu.
Proyek masih dalam tahap pengembangan dan akan segera tersedia di GitHub.

Dashboard - Report

Proyek ini merupakan visualisasi data interaktif yang dibangun menggunakan Microsoft Power BI untuk menganalisis data penjualan game online secara global dan per negara. Dataset diambil dari situs Kaggle dan mencakup informasi seperti total penjualan global, penjualan berdasarkan negara, serta tren penjualan dalam rentang waktu tertentu.
Dashboard ini dilengkapi dengan elemen visual seperti grafik batang, peta interaktif, dan filter dinamis agar pengguna dapat mengeksplorasi data berdasarkan tahun, wilayah, maupun kategori lainnya.
